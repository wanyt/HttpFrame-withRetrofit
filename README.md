# Http_withRetrofit

> 项目处于维护阶段

## 项目使用到的技术

* Retrofit请求网络
* Material控件CollapsingToolbarLayout
* Retrofit下载，上传文件（待添加）
* 在调试模式下查看最近请求和响应（待添加）
* mvp架构（待添加）
